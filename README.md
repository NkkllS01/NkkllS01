# Personal Portfolio (Static vCard Style)

已清理旧内容并统一为静态个人主页模板（无 Hugo 代码依赖）。

## 文件结构

- `index.html`：页面结构与可替换内容占位
- `assets/css/style.css`：样式
- `assets/js/script.js`：标签切换、作品筛选、年份与侧栏折叠逻辑

## 本地预览

```bash
python3 -m http.server 8080
```

打开：<http://localhost:8080>

## 下一步（你只需要替换占位符）

把 `index.html` 中形如 `[你的姓名]`、`[项目链接]`、`[文章标题]` 的占位内容替换为你的真实信息即可。
